## 🛠️ How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/kinshukkush/AAMS/issues) to see if the bug has already been reported.
*   If not, open a new issue with a clear title, description, and steps to reproduce.

### Suggesting Enhancements
*   Open an issue with the "enhancement" label.
*   Describe the feature you'd like to see and why it would be useful.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  Ensure your code follows the existing style guidelines.
4.  Issue that pull request!
